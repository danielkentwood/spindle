# Spindle Functional Documentation

This folder is the current functional documentation for the `spindle-kg` package.

## Start Here

- `QUICKSTART.md`: install, run a first extraction, and start the API.
- `ENV_SETUP.md`: required and optional environment variables.
- `PACKAGE_STRUCTURE.md`: what each package module does.
- `CONFIGURATION.md`: `SpindleConfig`, Hydra config groups, and pipeline wiring.
- `TESTING.md`: test matrix and common `uv` commands.

## Core Runtime Docs

- `kos_data_model.md`: KOS runtime model, files, indices, and APIs.
- `provenance_model.md`: provenance storage model and joins.
- `GRAPH_STORE.md`: graph persistence and query surface.
- `ENTITY_RESOLUTION.md`: resolver pipeline and expected inputs.
- `OBSERVABILITY.md`: structured events and persistence.

## Reference Docs

- `glossary.md`: current terminology and component definitions.
- `pipeline_steps.md`: current stage flow and execution model.
- `user_stories_supported.md`: workflows currently supported by package and API.
